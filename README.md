# 📊 Prazo Manager

Sistema web para gerenciamento de contratos e prazos de vencimento, com foco em organização, visualização clara e priorização automática de tarefas.

## 🚀 Sobre o projeto

O Prazo Manager foi desenvolvido para facilitar o controle de contratos e seus respectivos vencimentos, permitindo identificar rapidamente itens críticos, próximos do vencimento e ativos.

A aplicação conta com um dashboard inteligente que calcula automaticamente os dias restantes e classifica cada registro com base na sua urgência.

---

## ⚙️ Funcionalidades

### 📊 Dashboard
- Visão geral dos registros
- Cards de resumo (total, vencidos, vencendo e ativos)
- Lista priorizada automaticamente por urgência

### 🧠 Lógica automática
- Cálculo dinâmico de dias restantes até o vencimento
- Classificação automática:
  - 🔴 Vencido
  - 🟡 Vencendo
  - 🟢 Ativo

### 🎯 Filtros interativos
- Filtrar registros por status
- Atualização dinâmica da interface

### 🔀 Ordenação inteligente
- Ordenação por nível de prioridade
- Itens mais críticos aparecem primeiro

### 🧭 Navegação
- Sistema de rotas com múltiplas páginas
- Estrutura pronta para expansão do sistema
